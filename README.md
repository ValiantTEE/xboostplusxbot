# XBoostPlus Telegram Bot

XBoostPlus is a Telegram bot that helps users automatically boost their tweets and buy real X (formerly Twitter) followers. It provides subscription packages, followers packages, and a seamless payment system via Ethereum, BNB, or Solana. Admins can monitor and mark orders as completed directly through Telegram.

---

## Features

- Automatic tweet engagement boosts via subscription tiers
- Buy real followers packages (Blue Tick and standard accounts)
- Add and manage multiple X accounts
- Payment via Ethereum, BNB Smart Chain, or Solana
- Admin panel to view and complete orders
- Asynchronous database with SQLite (aiosqlite)
- Inline keyboard navigation for a smooth user experience
